# 설명
- wbmMC 라이브러리 사용법 문서
- paper기반의 API를 기반으로 제작됨



# 플러그인에 적용 방법
1. 플러그인 [다운로드]
2. eclipse에서 프로젝트 우클릭 
3. build path 클릭 
4. Add External JARs... 클릭 
5. wbmMC 클릭
6. 플러그인 제작
7. 버킷의 plugins 폴더에 wbmMC.jar 넣고 실행
8. console에 `wbmMC ON` 글자가 출력되면 성공
